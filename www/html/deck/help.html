Fastest Ways In

Deck Terminal Run deck, then choose 🎛️ sdr.
Browser Audio Use /sdr/ when you want sound on your phone or laptop instead of the Pi.
Cheatsheet The launcher already links to the SDR cheatsheet if you want the command-heavy version.

Audio Reality Check

If you tune SDR stuff from plain SSH, the sound plays out of the Pi's local audio output, not back through your terminal.

Use the browser page at /sdr/ when you want remote audio on the device in your hand.

SSH = control only. Browser = actual remote listening.

SDR Menu Map

🧭 SDR Snapshot Quick state check. Tells you if the dongle is even there and sane.
🧪 Self Test Opens the tuner and proves the SDR stack can actually sample.
🌐 Remote Stream Shows the web tuner URLs and the current remote stream state.
📻 FM Presets / 🎛️ Tune FM Music radio. Presets are the easy path. Tune FM is for custom frequencies.
🛫 Airband Listen Aircraft voice. Good weird-radio starter mode.
🚓 Utility / Public Safety Listen Analog narrowband listening only. Not magic for encrypted or trunked systems.
🌡️ 433MHz Hunt Finds sensors, remotes, cheap gadgets, weather junk, random small devices.
📟 Pager Hunt Looks for pager traffic and decodes what it can.
✈️ Plane Radar ADS-B data mode. Not audio. Good for aircraft position tracking.

Good Starter Moves

Just want sound fast? Open /sdr/, hit an FM preset, and listen in the browser.
Want planes? Use ✈️ Plane Radar in the launcher or run ~/pibulus-os/scripts/sdr_lab.sh planes.
Want weird signals? Try 🌡️ 433MHz Hunt first. It gives the fastest "holy shit" results.
Want voice traffic? Try 🛫 Airband Listen or analog utility channels before expecting police scanners from 1998.

If It Feels Broken

Run these in order and you will usually know what kind of broken it is:

~/pibulus-os/scripts/sdr_lab.sh status
~/pibulus-os/scripts/sdr_lab.sh selftest
~/pibulus-os/scripts/sdr_lab.sh remote-status
systemctl status sdr-remote.service --no-pager

What you want: dongle detected, R820T tuner visible in self-test, and the remote service marked active.

Useful Truths

The Pi is not busy all the time The SDR stream wakes up only when you use it, then auto-sleeps after a few idle minutes.
Favorites live in your browser The remote SDR page saves favorites locally on that device, not globally on the Pi.
Modern public safety is messy If it is digital, trunked, or encrypted, the cheap easy path ends fast.

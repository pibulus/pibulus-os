What is this place?

Quick Cat Club is a self-hosted server running on a Raspberry Pi 5 in Melbourne. It runs pirate radio, a film library, a book collection, retro games, graphic novels/comics, Palestine resources, interactive fiction adventures, music collection, an offline copy of Wikipedia and more.. No cloud. No tracking. No subscriptions. Just a hard drive and some docker containers doing their best.

Why does this exist?

Because the internet used to be weird and personal and ours. Before everything became five websites full of screenshots of the other four. This is a tiny corner of the web that belongs to no platform, answers to no algorithm, and doesn't need your data to function.

Will this server go down?

Almost certainly. It's a 4GB Raspberry Pi running 17 containers strapped to a hard drive. It crashes when you look at it funny. But it always comes back.

Is my data being collected?

No. There's no analytics, no cookies, no tracking pixels, no third-party scripts. The server doesn't know who you are and doesn't want to.

What's the philosophy here?

Software is politics. Every platform you use is a statement about who owns your attention. This server runs on the idea that tools should help people and then leave them alone. Small is beautiful. Can't scale is a feature.

Can I build something like this?

Absolutely. A Raspberry Pi 5, a hard drive, and a weekend. Docker makes it modular. The whole setup is open source.

How can I contribute?

Got music? Books? Comics? ROMs? Weird knowledge? Send it. There's an upload link up above. Leave a message if you'd lke or make a request and I'll try and get it into the server for ya.

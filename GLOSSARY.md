# 📖 CYBERDECK GLOSSARY

- **Quick Cat Club:** The underground identity of this deck and its community.
- **Golden Image:** A compressed archive of the system's logic and configuration.
- **Bunker Mode:** A state of maximum security (key-only access, all stacks down).
- **Travel Mode:** A state allowing password access for remote SSH logins.
- **Media Scavenger:** The integrated tool for yt-dlp and Soulseek downloads.
- **Passport:** The physical 2TB drive that holds the deck's soul.
